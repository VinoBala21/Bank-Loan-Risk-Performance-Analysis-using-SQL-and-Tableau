
📊 Bank Loan Risk & Performance Analysis using SQL and Tableau
🔍 Project Overview

This project analyzes bank loan data to evaluate lending performance, identify high-risk borrowers, and generate meaningful business insights. The analysis is performed using SQL for data querying and Tableau for interactive data visualization.

🎯 Objectives

Analyze overall loan performance and key business KPIs
Identify factors contributing to loan defaults
Segment borrowers based on risk (income, DTI, loan grade)
Provide insights to improve lending decisions
🛠️ Tools & Technologies
SQL (MySQL)
Tableau
CSV Dataset

📁 Dataset

The dataset contains financial loan records including:

Loan amount
Interest rate
Loan status (Fully Paid, Current, Charged Off)
Borrower income
Employment length
Loan purpose

Dataset file: Bank_loan_cleaned.csv

📊 Key Analysis Performed

Overall lending performance (Total Applications, Funded Amount, Revenue)
Good vs Bad loan classification
Monthly loan trends
Regional (state-wise) analysis
Default rate by loan purpose and employment length
Risk analysis based on income, DTI, and loan grade

🚀 Conclusion

This project demonstrates how SQL and Tableau can be used together to analyze financial data, identify risk patterns, and support better business decision-making in the banking sector.
